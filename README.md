# Image Slider
1. Go ahead an _clone_ this repo with
```
git clone https://github.com/Confidence-Okoghenun/Image-Slider.git
```
2. Install the packages with npm or yarn

```
npm i
#or
yarn install
```
3. Start sass with 
```
npm run sass
#or
yarn sass
````
